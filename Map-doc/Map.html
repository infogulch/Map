Map()

Map(func, arr_arg1 [, ...arr_argN])

Parameters

func The function to call
arr_arg1..arr_argN Must be an array. Arrays of args to pass to func. The position of the arr_arg passed to Map() is the same position passed to func.

Returns

An array of the results.
